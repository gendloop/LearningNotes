# Summary

## 安装

```bash
scoop install extras/vscode
reg import "<root>\install-context.reg"
reg import "<root>\install-associations.reg"
```

## 常用快捷键

* 打开文件夹   `Ctrl+K, Ctrl+O`
* 资源管理器   `Ctrl+Shift+E`
* 搜索        `Ctrl+Shift+F`
* 源代码管理   `Ctrl+Shift+G`
* 运行和调试   `Ctrl+Shift+D`
* 扩展        `Ctrl+Shift+X`
* 命令行      `Ctrl+Shift+P`
* 输出面板    `Ctrl+Shift+U`
* 问题面板    `Ctrl+Shift+M`
* 新建文件    `Ctrl+N`
* 保存文件    `Ctrl+S`
* 运行        `F5`
* 缩小        `Ctrl+-`
* 放大        `Ctrl+=`
* 多行编辑    `Ctrl+Shift+L`
