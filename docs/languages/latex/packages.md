# Packages

常见的LaTex包.

## graphicx

```latex
\graphicspath{{figures/}}
\includegraphics
```

## subcaption

```latex
\begin{subfigure}{0.4\textwidth}
```

## float

```latex
\begin{figure}[H]
```

## wrapfig

```latex
\begin{wrapfigure}{l}{0.25\textwidth}
```

## amsmath

```latex
\begin{equation}
\begin{equation*}
```

## babel

```latex
\babel[english, french]{babel}
```

## ctex

支持中文
