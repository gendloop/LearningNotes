# Weixin Official Account

微信公众号平台 <https://mp.weixin.qq.com/>

## 创建公众号

1. 进入[申请公众号网页](https://mp.weixin.qq.com/cgi-bin/loginpage?t=wxm2-login&lang=zh_CN)
2. 点击 `立即注册`
3. 选择 `订阅号`
4. 填写基本信息
    1. 邮箱 `xxx`
    2. 激活邮箱
    3. 密码 `xxx`
    4. 重复密码 `xxx`
5. 选择类型
    1. `中国大陆`
    2. `确定`
    3. `订阅号`
    4. `选择并继续`
6. 信息登记
    1. 主体类型 `个人`
    2. 身份证姓名 `xxx`
    3. 身份证号码 `xxx`
    4. 扫码验证管理员身份
    5. 管理员手机 `xxx`
    6. 短信验证码 `xxx`
    7. 创作平台 `其他` `语雀`
    8. 创作者昵称 `gendloop`
    9. ~~证明资料~~
    10. 证明链接 <https://www.yuque.com/gendloop>
7. 公众号信息
    1. 账号信息 `gendloop`
    2. 功能介绍 `记录 & 分享`
    3. 内容类目 `科技/互联网, 其他`
    4. 运营地区 `国家` `省份` `城市`
    5. `完成`
